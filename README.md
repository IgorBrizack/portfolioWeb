# Igor Brizack — Portfolio

Personal portfolio built with React 18, TypeScript, Vite and Tailwind CSS. Bilingual (EN/PT) via `react-i18next`, animations via Framer Motion.

## Stack

- **Build**: Vite
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animation**: Framer Motion
- **i18n**: react-i18next (English / Portuguese)

## Development

```bash
npm install
npm run dev      # dev server
npm run build    # typecheck + production build to dist/
npm run preview  # preview production build locally
```

## Content

All copy lives in `src/i18n/locales/en.json` and `src/i18n/locales/pt.json`. Structured data (skills list, social links, resume paths) lives in `src/lib/data.ts`.
